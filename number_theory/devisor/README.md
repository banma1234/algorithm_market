## 제곱근을 이용한 약수 구하기

target을 `1`부터 target의 `제곱근`까지 나누었을 때 0으로 나누어 떨어진다면 target을 `나눈 수`와 그 `몫`은 `약수`이다.

---

100의 제곱근 = 10
1 ~ 10까지 100의 약수 = [1, 2, 4, 5, 10]

- 100 / `1` = `100`
- 100 / `2` = `50`
- 100 / `4` = `25`
- 100 / `5` = `20`
- 100 / `10` = `10`

100의 약수 = [1, 2, 4, 5, **`10`**, **`10`**, 20, 25, 50, 100]

위와같이 나눈 수와 몫이 겹치는 경우가 있기 때문에 `Set` 자료형을 이용하여 중복값을 소거해준다.
